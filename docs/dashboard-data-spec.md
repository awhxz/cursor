# ТЗ: перенос задач из Google Sheets в дашборд

## 1. Источник

- Источник — Google Sheets `1HB29ZDJvyPuyfG5z4MguTEhJ4_OngRqDVRxKfqV9rWQ`.
- Для раздела «Задачи в аналитике» используется лист «Актуальные».
- Дашборд читает только первые 9 столбцов (`A:I`). Недельные столбцы справа не загружаются.
- Сопоставление выполняется по нормализованному тексту заголовка, а не только по жесткому номеру столбца. Это защищает интеграцию от небольших изменений названий.

## 2. Столбцы и правила

| Позиция | Поле | Перенос в дашборд | Фильтр/сортировка |
| --- | --- | --- | --- |
| A | Тикет | Значение должно быть URL, начинающимся с `http://` или `https://`. В дашборде показывается кликабельная ссылка. Невалидное значение показывается красным и не становится ссылкой. | Сортировка |
| B | Название | Текст без дополнительного преобразования. | Поиск и сортировка |
| C | Заказчик | Выводится фактическое значение из Google Sheets. Список не дублируется в коде. | Динамический фильтр и сортировка |
| D | Ответственный | Выводится фактическое значение. | Динамический фильтр и сортировка |
| E | К чему относится | Выводится фактическое значение. | Динамический фильтр и сортировка |
| F | Ценность | Значения `1`, `2`, `3`, `4` подсвечиваются красным, оранжевым, желтым и зеленым. Пустое или иное значение — красным. | Числовая сортировка |
| G | Время в аналитике | Та же шкала `1`–`4`, что и для «Ценности». | Числовая сортировка |
| H | Статус | Выводится как метка. Статус определяет лист Google Sheets и будущий раздел дашборда. | Динамический фильтр и сортировка |
| I | Комментарий | Текст без преобразования. | Поиск и сортировка |

## 4. Интерфейс

- Первая вкладка — «Задачи в аналитике».
- Фильтры: заказчик, ответственный, «к чему относится», ценность, время в аналитике, статус и общий текстовый поиск.
- Значения фильтров строятся из фактически загруженных строк.
- Каждый из 9 столбцов таблицы сортируется по клику на заголовок. Повторный клик меняет направление.
- В верхней части остаются ссылка на исходную Google Sheets, время обновления и кнопка ручного обновления.
