# ТЗ: перенос задач из Google Sheets в дашборд

## 1. Источник

- Источник — Google Sheets `1HB29ZDJvyPuyfG5z4MguTEhJ4_OngRqDVRxKfqV9rWQ`.
- Для раздела «Задачи в аналитике» используется лист «Актуальные» с `gid=1654701914`.
- Дашборд читает только первые 9 столбцов (`A:I`). Недельные столбцы справа не загружаются.
- Парсер ищет строку заголовков по известным алиасам в первых 25 строках, поэтому служебные строки над таблицей не сдвигают маппинг.
- Сопоставление выполняется по нормализованному тексту заголовка, а не только по жесткому номеру столбца. Это защищает интеграцию от небольших изменений названий.
- До добавления display-fallback строка проходит валидацию: должно быть заполнено название или тикет формата `BS-12345`/HTTP(S) URL. Служебные и пустые строки отбрасываются до построения фильтров и групп.

## 2. Столбцы и правила

| Позиция | Поле | Перенос в дашборд | Фильтр/сортировка |
| --- | --- | --- | --- |
| A | Тикет | Если значение начинается с `http://` или `https://`, оно показывается как кликабельная ссылка. Любое другое значение передается из Google Sheets как обычный текст, без подсветки ошибки. | Сортировка |
| B | Название | Текст без дополнительного преобразования. | Поиск и сортировка |
| C | Заказчик | Выводится фактическое значение из Google Sheets. Список не дублируется в коде. | Динамический фильтр и сортировка |
| D | Ответственный | Выводится фактическое значение. | Динамический фильтр и сортировка |
| E | К чему относится | Выводится фактическое значение. | Динамический фильтр и сортировка |
| F | Ценность | Значения `1`, `2`, `3`, `4` подсвечиваются красным, оранжевым, желтым и зеленым. Пустое или иное значение — красным. | Числовая сортировка |

| I | Комментарий | Текст без преобразования. | Поиск и сортировка |

## 4. Интерфейс

- Вкладки: «Актуальные», «Бэклог», «Готовые и стоп», «Все». Они работают одновременно с поиском и фильтрами.
- Заказчик, ответственный, «к чему относится» и статус фильтруются через мультиселекты. Внутри одного фильтра значения объединяются по OR, а разные фильтры — по AND. Каждый dropdown содержит tri-state «Выбрать все» и не закрывается после выбора одного пункта.
- Значения фильтров строятся из фактически загруженных строк.
- Каждый из 9 столбцов таблицы сортируется по клику на заголовок. Повторный клик меняет направление.
- Основная таблица группирует задачи по ответственным. В отдельный блок «Задачи в очереди» выносятся только задачи со статусом «В очереди» без назначенного ответственного. Задача «В очереди» с ответственным остается в его группе в основной таблице.
- В верхней части остаются ссылка на исходную Google Sheets, время обновления и кнопка ручного обновления.
